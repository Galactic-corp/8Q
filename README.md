Computer with built-in General Purpose 8 Qubit Optical Quantum Processor

## Features
![Feature List](./docs/source/imgs/8q.png)

* Quad-Core Cortex-A53
* Mali-T720 MP2 GPU
* 1x DDR4 Ram Slot (Max 64GB)
* 32/64 Bit Compatible
* Linux Capable
